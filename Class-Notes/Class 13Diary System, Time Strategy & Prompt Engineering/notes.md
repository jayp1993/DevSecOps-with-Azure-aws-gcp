# 1. Diary System

A Diary System helps manage daily life, work, learning, and goals in a disciplined way.  
It improves focus, planning, and productivity.

---

## Morning Planning

Morning planning means preparing your tasks before starting the day.

### Steps
- Wake up and prepare mentally for the day.
- Write all important tasks in a diary/notebook.
- Decide task priorities.
- Set deadlines for tasks.
- Identify urgent and important work.

### Example
| Time | Task |
|------|------|
| 7 AM | Exercise |
| 9 AM | Office Work |
| 6 PM | Study DevOps |
| 9 PM | Revision |

---

## Time Blocking

Time Blocking means dividing the day into multiple blocks for specific tasks.

### Advantages
- Reduces distractions
- Improves concentration
- Prevents multitasking confusion
- Increases productivity

### Example
| Time Block | Activity |
|------------|-----------|
| 7–8 AM | Exercise |
| 9–1 PM | Office Work |
| 2–4 PM | Azure Practice |
| 5–6 PM | Break |
| 7–9 PM | DevOps Learning |

---

## Night Review

At night, review your entire day.

### Check:
- Which tasks were completed?
- Which tasks are pending?
- Where was time wasted?
- What improvements are needed tomorrow?

### Benefits
- Self-improvement
- Better planning
- Discipline development
- Confidence improvement

# 3. ABCDE Time Management Technique

ABCDE Technique is used for task prioritization.

---

## A – Very Important Tasks

These are the highest priority tasks.

### Examples
- Production issue fixing
- Client meeting
- Important project submission

### Rule
Complete A category tasks first.

---

## B – Important but Not Urgent

These tasks are important but can wait slightly.

### Examples
- Learning new technology
- Documentation
- Skill development

---

## C – Nice to Do

Tasks that are optional.

### Examples
- Watching random videos
- Casual social media usage

---

## D – Delegate

Tasks that can be assigned to others.

### Examples
- Report formatting
- Simple monitoring work

---

## E – Eliminate

Completely remove useless activities.

### Examples
- Excessive scrolling
- Time-wasting discussions
- Unnecessary meetings

---

# Steps to Implement ABCDE Technique

## Step 1
Write all tasks in your diary.

## Step 2
Assign A, B, C, D, or E labels.

## Step 3
Start with A category tasks.

## Step 4
Complete B and C tasks afterward.

## Step 5
Delegate D category tasks.

## Step 6
Eliminate E category tasks completely.

---

# 7. Prompt Engineering

Prompt Engineering is the skill of writing effective AI prompts.

---

# What is a Prompt?

A prompt is an instruction/input given to an AI model.

---

# Prompt Example

## Real Life Example

Sugarcane → Machine → Juice

Similarly:

Input(Prompt) → AI/ML Model → Output

---

# Prompt Engineer

A Prompt Engineer is a person who can:
- Write smart prompts
- Get better AI outputs
- Improve AI efficiency
- Save time

---

# Good Prompt Characteristics

A good prompt should be:
- Clear
- Specific
- Detailed
- Structured

---

# Example of Bad Prompt

"Explain Azure"

---

# Example of Good Prompt

"Explain Azure Virtual Machine with architecture, use cases, pricing model, and production scenario."
